## 🎯 HVORDAN OPPDATERE RESTERENDE FILER

For hver fil:

1. Åpne filen i VS Code
2. Trykk `Ctrl+F` og søk etter: `<button`
3. For hver button som ikke er disabled eller veldig liten:
   ```jsx
   // FØR:
   <button className="px-4 py-2 bg-purple-600">
   
   // ETTER:
   <button className="ripple-effect px-4 py-2 bg-purple-600">
   ```

4. Lagre filen

**OBS:** Ikke legg til ripple-effect på:
- Disabled buttons
- Icon-only buttons som er < 32px
- Buttons inne i inputs (f.eks. search icons)
